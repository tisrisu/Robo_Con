# Robo_Con

Machine Learning project for symbol classification.

## Files
- `prepare_data.py` - Data preprocessing and augmentation
- `train_model_v2.py` - Model training script
- `garbage.py` - Testing script
- `.gitignore` - Git ignore file
- `final_model.py` - final model script

## Usage
1. Prepare data: `python prepare_data.py`
2. Train model: `python train_model.py`
3. Test model: `python garbage.py`
4. final model: `final_model.py`
